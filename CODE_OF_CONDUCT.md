# Code of Conduct

This project follows the [Contributor Covenant, version 2.1][cc21], a
widely-adopted code of conduct for open source communities.

The full text is available at:
<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

By participating in this project (issues, pull requests, discussions,
or any project space) you agree to abide by its terms.

## Reporting violations

Report unacceptable behaviour through one of the following channels:

- **Private security advisory**: open a private report via
  <https://github.com/daniele-chiappa/gosidian/security/advisories/new>
  (also used for security issues; the form lets you mark the report
  as a conduct concern in the title).
- **GitHub issue**: only for non-sensitive matters; use a clear title
  prefixed with `[conduct]` and the maintainer will move the discussion
  to a private channel before responding substantively.

Reports are reviewed by the project maintainer
(GitHub: [@daniele-chiappa][maint]). The reporter's identity is kept
confidential where possible. Action follows the **Enforcement
Guidelines** of the Contributor Covenant 2.1 — Correction → Warning →
Temporary Ban → Permanent Ban — applied in proportion to the impact
and recurrence of the behaviour.

## Scope

This Code of Conduct applies within all project spaces (repository
issues, pull requests, discussions, code reviews) and when an
individual is officially representing the project in public spaces.
